# Kumbara-Kala — State Pride 🏺

GenAI powered Android app for clay artisan digital branding

## Problem Statement
Potters are losing business to plastic and steel because they 
cannot market the health benefits of clay to modern buyers.

## Features
- Artisan Profile — Name, Village, Phone
- Clay Product Catalog — 12 products  
- AI Story Card Generator — Google Gemini API
- Canvas Card Rendering — 1080x1080px
- WhatsApp Share — One tap sharing

## Technologies Used
Kotlin, Android Studio, Google Gemini API, OkHttp, 
Android Canvas, Bitmap, RecyclerView, Share Intent, 
FileProvider, SharedPreferences

## How to Run
1. Clone this repository
2. Open in Android Studio
3. Get free API key from aistudio.google.com/apikey
4. Replace YOUR_API_KEY_HERE in StoryCardActivity.kt
5. Run on Android phone API 24 and above
